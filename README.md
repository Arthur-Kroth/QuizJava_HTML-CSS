# Java Quiz Application

Interactive front-end project developed to demonstrate HTML5 semantic structure, form elements, accessibility concepts, and CSS styling through a Java programming quiz.

## Overview

This project simulates an educational quiz platform where users can test their knowledge of Java fundamentals through different types of form interactions.

It was built to practice:

- Semantic HTML structure
- Form components and input types
- CSS layout and styling
- User interaction design
- Accessibility fundamentals

---

## Features

- Interactive Java quiz with 8 questions
- Multiple input types:
  - Radio buttons
  - Text input
  - Password field
  - Date selector
  - Checkboxes
  - File upload
  - Dropdown menu
  - Image-based question

- Scoring table
- Expandable answer section
- Feedback submission form
- Styled responsive layout using Flexbox

---

## Tech Stack

**Frontend**
- HTML5
- CSS3

**Concepts Applied**
- Semantic tags (`header`, `main`, `section`, `footer`)
- Forms and validation-ready structure
- Flexbox layout
- Custom styling with reusable selectors
- Accessibility elements (`label`, `fieldset`, `legend`, `abbr`, `figcaption`)
